TankVenn

Open Data

Community-reported fuel station and price data for Norway, updated every 12 hours. Free to use for any purpose.

Stations

All fuel stations in Norway with brand, address, and coordinates.

Prices

Current fuel prices (Petrol 95, Petrol 98, Diesel) with timestamps and report counts.

prices.json

Data is exported from TankVenn's Firestore database via a GitHub Actions workflow. Station data is sourced from OpenStreetMap. Prices are reported by app users.
